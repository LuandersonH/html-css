NEGRITO E ITÁLICO

Há duas formas de se fazer uma palavra em negrito!

O Negrito semântico (com significado) é através da tag: <strong>
O Negrito não-semântico (para design) é através da tag: <b>

Há duas formas de se fazer uma palavra em itálico!

O Itálico semântico é através da tag: <em>
O Itálico não-semântico é através da tag: <i>

FORMATAÇÕES ADICIONAIS

Para usar um marca texto, a tag <mark>

Para deixar a letra pequena, a tag: <small>

Para deixar grande, CÓDIGO NÃO USÁVEL: <big>

Para sugerir que a palavra foi deletada: <del>

Para indicar que uma palavra foi inserida: <ins>

Sublinhado sem semântica, para design: <u>

Para colocar o n°/palavra em cima: <sup>

Para colocar a n°/palavra embaixo: <sub>

CITAÇÕES E CÓDIGOS

Para colocar uma fonte monoespaçada (ideal p códigos): <code>

Para mostrar o texto exato do código (quebra de linha): <pre>

Para indicar que estou citando algo (""): <p>

Para citar um parágráfo todo (formatação automática):

<blockquote>

Para indicar que algo foi citado no blockquote (mecanismos): <blockquote cite="link do livro, blog...">

Para aparecer um texto ao passar o mouse na palavra: <abbr title="legenda">

A tag inútil de fazer o texto pro lado contrário (Desse Jeito): <bdr dir ="ltr"/"rtl">
